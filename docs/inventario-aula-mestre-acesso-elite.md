# Aula Mestre do Acesso Elite

## Objetivo

Curso obrigatorio e inicial do Acesso Elite, criado para ensinar o aluno a navegar pela estrutura da plataforma antes de abrir os demais cursos.

## Estrutura

### Modulo 1 - Aula Mestre

1. Como navegar pela plataforma
   - Video: definir no editor
   - Observacao: aula inicial obrigatoria antes dos demais cursos do Elite

## Regras

- Ordem no Acesso Elite: `0`
- Liberacao no Elite: `0 dias`
- Curso oculto do catalogo publico
- Obrigatorio antes dos demais cursos do Elite
